## Создать ветку

```sh
git checkout -b'переход в неё' <название ветки>
```

## Посмотреть список веток в репозитории

```sh
git branch
```

## Создать новую ветку

```sh
git branch <название ветки>
```

## Удалить ветку

```sh
git branch -d <название ветки>
```
